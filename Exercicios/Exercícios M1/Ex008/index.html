Nesta frase temos um Termo em itálico utilizando a tag <em> (semântico)

Texto Marcado

Nesta frase temos um Termo marcado utilizando a tag <mark>
Aqui temos um Termo marcado com cor diferente utlizando a tag <mark style="background-color"> (Utilizando CSS(tag style)) (para mudar a cor em todos os "marks" colocar no head do codigo) 


    <head>   
        <style>
           mark { 
            background-color
           }
        </style>
    </head>
